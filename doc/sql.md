## 二、函数

### 2.1、CONCAT

使用方法：
**concat(str1,str2,…)** 

返回结果为连接参数产生的字符串。如有任何一个参数为NULL ，则返回值为 NULL。

```
mysql> select concat('11','22','33');
+------------------------+
| concat('11','22','33') |
+------------------------+
| 112233 |
+------------------------+
1 row in set (0.00 sec)

MySQL的concat函数在连接字符串的时候，只要其中一个是NULL,那么将返回NULL
mysql> select concat('11','22',null);
+------------------------+
| concat('11','22',null) |
+------------------------+
| NULL   |
+------------------------+
1 row in set (0.00 sec)
```
